게시글 세부 내역 조회
SELECT * FROM POST WHERE delete_yn = 'N' AND idx = ? limit paginationInfo.firstRecordIdx, recordsPerapge

게시글 내역 업데이트
update post set update_time = now(), content = ?, writer = ?, notice_yn = ?, secret_yn = ? where idx = ?

게시글 삭제
update post set delete_yn = 'N', delete_time = now() where idx = ?

게시글 목록 조회
